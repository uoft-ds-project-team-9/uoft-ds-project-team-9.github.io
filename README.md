# SCS3250 Group 9 Final Project

## Project Title
Major Crime indicators in Toronto 

## Description 
To Analyze a dataset that contains major crime indication for each neighborhood in Toronto.

## Objectives
- Visualization of major crime rates over months/years in each neighborhood division
- Which is the most safe (lowest crime rate) division in Toronto
- Is there relation between crime rate and season/ day of week
- ...

## Dataset
http://data.torontopolice.on.ca/datasets/mci-2014-to-2019/data
